# Node-Macro

A sophisticated Python-based Minecraft macro application with advanced PvP features

## Features

- **Dual Button Support**: Control both left and right mouse clicking independently
- **Customizable CPS**: Set minimum and maximum clicks-per-second for natural variation
- **Multiple Activation Modes**:
  - **Hold Mode**: Click while key is pressed
  - **Toggle Mode**: Toggle clicking on/off with key presses
- **Custom Key Binding**: Assign any keyboard key to activate clicking
- **Real-time Status**: Visual indicators show active/inactive states
- **Threaded Execution**: Smooth operation without GUI freezing
